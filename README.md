# Get Started

1. Install [node](https://nodejs.org/en/download/package-manager/)
2. Install [yarn](https://yarnpkg.com/lang/en/docs/install/)
3. Then:

```
git clone git@github.com:carlos-peru/next-with-api.git
cd next-with-api
yarn install
yarn run dev
```
